### **Installation**  

+ via **CDN**  
``` html
  <script src="youpath/jroll.js"></script>  
  <script src="https://unpkg.com/vue-jroll-scroller-plugin/dist/vue-jroll-scroller-plugin.min.js"></script>
```  

+ via **NPM**  
```bash
$ npm install jroll --save  
$ npm install vue-jroll-scroller-plugin --save
```
